Install Root on Windows

准备

  • 下载root5.3.4-36 没有测试更高版本
  • 安装python2.7 更高版本会导致问题
  • 安装cmake3.2.2 更高版本会导致问题
  • 安装Visual Studio 12 2013 更高版本会导致问题

生成Visual Studio .sln

cmake -G “Visual Studio 12 2013” path/to/root/source -Droofit=ON
-Droofit=ON means the libRooFit will be built

问题

  • 第一次cmake会失败，再试一次就好了
  • gunichartables.h 需要保存为utf8-with-signature

打包

编译PACAGE项目即可
