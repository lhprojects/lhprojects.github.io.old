博彩


我们研究一下博彩问题。

假设有两支队伍比赛，比如说A队和B队，投注者可以投注其中一只队伍。无论投注者所投的队伍输赢，投注者都要首先付出一定的本金。如果投注者投注的队伍赢了，投注者这可以获取一定的回报；如果投注者投注的队伍输了，那么投注者没有回报。每投入一元的回报就是所谓的赔率。一般两队的赔率并不相同，赢面比较大的队伍赔率比较低，而且赢面比较小的队伍赔率比较低。所谓低风险低收入，高风险高收入。赔率都是大于1的，否则回报不抵成本。

博彩公司的策略

对于博彩公司而言，首要的是增加收益，降低风险。

最小化风险

在我们的研究里，假设降低风险是首要的。假设买A队赢的资金为 $N_A$ ，买B队赢的资金为 $N_B$ 。那么如果A队赢了，那么博彩公司正收益的条件为
$$
N_A O_A <= (N_A + N_B)
$$
或者说
$$
N_A / (N_A + N_B) <= 1/ O_A
$$
那么如果B队赢了，那么博彩公司正收益的条件为
$$
N_B O_B <= (N_A + N_B)
$$
或者说
$$
1 - 1/ O_B <= N_A / (N_A + N_B)
$$
联合两个公式可得
$$
1 - 1/ O_B <= N_A / (N_A + N_B) <= 1 / O_A
$$
如果以降低风险为首要条件，那么博彩公司首先应该估计$N_A / (N_A + N_B)$，然后定下赔率$O_B$和$O_A$。如果投注者实际的$N_A / (N_A + N_B)$与估计$N_A / (N_A + N_B)$的比较接近，都在$[1 - 1/ O_B, 1 / O_A]$区间内，那么博彩公司实际上是稳赚不赔的。如果$N_A / (N_A + N_B)$不在$[1 - 1/ O_B, 1 / O_A]$区间内，那么博彩公司要么大赚，要么大赔，这取决于到底那支队伍赢。博彩公司可以根据投注情况实时更改赔率，即由实时的$N_A / (N_A + N_B)$来调整$O_B$和$O_A$。

对于博彩公司，赔率不应设置的过高，否则区间过小，增大了出现风险的可能性；同时赔率不应设置的过低，否则对于投注者来说投注就不再有吸引力了。

我们从上式可以看出$1 - 1/ O_B <= 1 / O_A$，即$1/O_A + 1/O_B>1$。实际的博彩设置都是满足这个关系式的。

这里可以看到，赔率是通过投注者的投注者行为确定的。那么投注者的策略又是什么？

投注者的策略

我们知道那只队伍输赢有一定的或然性。假设队伍A赢得的概率是$P_A$，队伍B赢得的概率是$P_B$。那么投注A队伍的收益期望为$P_A O_A - 1$。如果$P_A > 1 / O_A$，那么就可以投注A队伍。投注B队伍的收益期望为$(1 - P_A) O_B - 1$。如果$P_A < 1 - 1 / O_B$，那么就可以投注B队伍。或者说如果$P_A$不在区间$[1 - 1/ O_B, 1 / O_A]$，那么就可以通过投注某一支队伍而获得正的收益期望。什么时候会出现这种投注者的机遇呢？

最大化收益

我们假设博彩公司是最小化风险的，而投注者是最大化收益的。我们知道$N_A / (N_A + N_B)$在区间$[1 - 1/ O_B, 1 / O_A]$里，同时要求$P_A$不在区间$[1 - 1/ O_B, 1 / O_A]$里。这要求$N_A / (N_A + N_B)$与$P_A$差别足够大，并且区间$[1 - 1/ O_B, 1 / O_A]$足够小。

区间$[1 - 1/ O_B, 1 / O_A]$足够小，意味着博彩公司克制的抽成，这很容易理解，一般这个数值在10%左右。

为了简单起见，我们假定$[1 - 1/ O_B, 1 / O_A]$足够小，那么 $1 - 1/ O_B = 1 / O_A = N_A / (N_A + N_B)$。我们知道博彩公司首先会设置一个初始赔率，然后根据投注情况设置更改赔率，这叫做市场赔率。那么$P_A$与$N_A / (N_A + N_B)$是什么关系呢？如果$P_A>N_A / (N_A + N_B)$那么投注A队是有有收益的，所以$N_A$就会增加，直到$N_A / (N_A + N_B)=P_A$。所以赔率最终是有市场平均认为的A队胜率决定的。

在博彩公司有抽成的情况下基本结论基本不变。如果市场平均判断水平很低，$N_A/(N_A+N_B)$与真实胜率$P_A$差别很大，区间$[1 - 1/ O_B, 1 / O_A]$覆盖了$N_A/(N_A+N_B)$，但是不覆盖$P_A$。那么投注者就出现了正收益的机会。不妨我们假设$P_A > N_A/(N_A+N_B)$，虽然$1/O_A$会比$N_A/(N_A+N_B)$略大（由博彩公司决定），但是我们再假定$P_A > 1 / O_A$，此时投注A队伍是有正收益的。如果市场平均判断水平很高，$N_A/(N_A+N_B)$与真实胜率$P_A$差别很小，区间$[1 - 1/ O_B, 1 / O_A]$覆盖了$N_A/(N_A+N_B)$，同时覆盖了$P_A$。那么无论投注哪支队伍，收益期望的都是负的，只有博彩公司收益期望是正的，而且是没有风险的。

$$
N_A + N_B - PA
